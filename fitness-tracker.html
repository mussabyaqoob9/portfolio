About the Project

The Fitness Tracker App is a cross-platform mobile solution built with React Native and Firebase. It helps users track workouts, monitor progress, and stay motivated with personalized goals. This project showcases my skills in mobile UI/UX, backend integration, and responsive design.

Visual charts and analytics to monitor progress over time.

Profile & Social

Project Overview

A modern, responsive API Gateway built with React.js for the frontend dashboard, Node.js for the backend server, and styled with Tailwind CSS for a clean, consistent design system.

The gateway features a responsive admin dashboard for monitoring API traffic, managing services, and configuring routing rules with a modern, accessible UI built with React components.

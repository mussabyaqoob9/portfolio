About Flutter

Flutter is an open-source UI toolkit by Google for building natively compiled apps for mobile, web, and desktop from a single codebase.

It enables fast development, expressive UIs, and near-native performance—perfect for cross-platform apps like this Travel Planning App.

About GraphQL

GraphQL is a query language for APIs and a runtime for executing queries. It lets clients request exactly the data they need—nothing more, nothing less.

In this Travel App, GraphQL ensures efficient data fetching and smooth integration with booking and itinerary APIs.
